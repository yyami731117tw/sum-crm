# 多元商會員管理系統 (CRM Dashboard)

## 系統概述
多元商會員管理系統是一個專業的客戶關係管理平台，提供完整的會員資料管理、關係人管理、投資履歷追蹤及合約管理等功能。

## 版本資訊
- **當前版本**：v1.3.4
- **更新日期**：2025/02/03
- **Node.js 要求**：18.x

## 功能特點
### 會員管理
- 會員資料管理
  - 基本資料維護
  - 會員狀態控制
  - 會員期限管理
  - 變更紀錄追蹤
- 關係人管理
  - 關係人資料同步
  - 關係網絡追蹤
- 投資履歷
  - 投資記錄
  - 投資分析

### 合約管理
- 合約資料管理
  - 基本資料維護
  - 合約狀態控制
  - 合約期限管理
  - 變更紀錄追蹤
- 檔案管理
  - 合約掃描檔上傳
  - 相關文件管理
- 付款資訊
  - 付款方式設定
  - 銀行帳戶管理

### 系統管理
- 人員管理
  - 管理員設置
  - 客服人員管理
  - 權限控制機制
- 系統日誌
  - 操作記錄查詢
  - 變更歷史追蹤

## 技術架構

- TypeScript
- Material-UI (MUI)
- 本地存儲 (localStorage/IndexedDB)

- 本地資料庫
  - 會員資料
  - 合約資料
  - 專案資料
  - 系統設定

  - 系統日誌集合

- 電子郵件服務

- 💾 資料備份機制

## 部署指南
1. **安裝依賴**
```bash
npm install
```

2. **啟動服務**
```bash
npm run dev
```
